# El Gasificador — Catálogo Web

## Proyecto
Catálogo web responsive para El Gasificador, una empresa especializada en repuestos de gas.
Desarrollado con Astro + Google Sheets como base de datos.

## Colores de marca
- Primario: #9F3737
- Secundario: #AD4F4F
- Acento: #033059
- Neutro: #F2EAEA
- Neutro soft: #DAE1E7

## Skills disponibles
- `using-superpowers` — Framework de skills, invocar siempre al inicio
- `writing-plans` — Crear planes de implementación
- `executing-plans` — Ejecutar planes paso a paso
- `verification-before-completion` — Verificar antes de declarar trabajo completo
- `systematic-debugging` — Debugging sistemático con búsqueda de causa raíz
- `ui-ux-pro-max` — Motor de búsqueda de diseño UI/UX
- `ui-styling` — Tailwind/theming y componentes
- `brand` — Gestión de marca y tokens de color
